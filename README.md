# Password Security System

## Description
Password Security System is a cybersecurity project that analyzes password strength and estimates how secure a password is against brute force attacks.

## Features
- Password strength analysis
- Entropy calculation
- Password breach detection
- Crack time estimation
- User-friendly GUI

## Technologies Used
- Python
- Machine Learning
- Tkinter GUI

## How to Run
1. Clone the repository
2. Install dependencies
3. Run main.py
